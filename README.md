# whiDB
An in-memory DB written in JS

## Running the code

Verified working on:
- Windows 10 (WSL2 w/ Ubuntu 20.04)
  - NodeJS v14.17.5
  - npm v6.14.14
- Mac OS X 10.12.6
  - NodeJS v12.22.6
  - npm v6.14.14

1. Checkout the repo
2. `npm ci` to install dependencies
3. `npm run test` to verify tests are passing
4. `npm run start` to interact with the in-memory DB
